# Opis aplikacji
Aplikacja napisana w celu ułatwienia sobie zdania przedmiotu
Podstawy Kryptografii. Obecna wersja (1.2) pozwala na rozwiązywanie
równań prowadzących do znalezenia kluczy
szyfrujących, oraz szyfrowania/deszyfrowania dowolnych
ciągów znaków.

# Co dodano względem wersji poprzedniej:

- Dodano obsługę błędów
- Umożliwiono uruchamianie solverów z poziomu aplikacji
- Dodano obsługę digramów (N^2)

# Jak wejść do aplikacji?
Aby uruchomić aplikację należy:
- pobrać folder
- wejść w cmd (Windows) lub terminal (Mac)
- dostać się do lokalizacji folderu za pomocą komendy -cd
- uruchomić aplikację komendą: python app.py (Windows) lub python3 app.y (Mac)
